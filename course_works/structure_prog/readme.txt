Для компиляции файла course_work.c через терминал необходима команда gcc -Wextra -Wall course_work.c -lm;
У пользователя должны быть установлены базовые библиотеки языка C. 
Полиномы следует вводить в следующем виде: коэффициент*х^степень+коэффициент*x-коэффициент. Слагаемые с нулевым коэффициентом могут быть пропущены, так же могут быть не указаны коэффициенты, равные единице.
Упрощена запись при 1-й и 0-й степенях, они могут как присутствовать, так и отсутствовать.
Исполняемый файл запускается с помощью команды ./a.out (для Linux)
1. interval - любой. polynomial - 0 — должно быть выведено сообщение Equation have infinity number of answers.
2. interval - любой. polynomial - любое одно число != 0 — должно быть выведено сообщение Equation doesn't have any answers.
3. interval - -10, 10. polynomial - 2*x+10 — должно быть выведено: -5.00000
4. interval - 0, 10. polynomial - 2*x+10 — должно быть выведено сообщение Can't find any answers, try another interval or polynomial.
5. interval - -10, 10. polynomial - 12*x^23+10*x^3-4*x^2+2*x-5 — должно быть выведено: -0.839901.
6. interval - 0, 3. polynomial - 4*x^2-5*x-12 — должно быть выведено: 2.466365.
